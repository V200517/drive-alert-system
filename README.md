#  Drive Alert System

A real-time driver drowsiness and distraction detection system.

---

## 🚀 Features

* Detects driver drowsiness
* Real-time alerts
* Distraction detection
* Safety-focused system

---

## 🛠 Tech Stack

* Java (Spring Boot)
* Computer Vision (OpenCV)
* JUnit (Testing)

---

## ▶️ How to Run

1. Open project in IDE (IntelliJ / Eclipse)

2. Run:
   DrivealertApplication.java

OR via terminal:
mvn spring-boot
